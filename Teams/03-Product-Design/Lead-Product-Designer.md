# Lead Product Designer (UI/UX)

## Position Details
- **Department:** Product & Design
- **Reports To:** CPO
- **Direct Reports:** Senior Product Designer(s), UX Researcher
- **Employment Type:** Full-time

## Role Summary
World-class UI/UX and visual design — portfolio at Dribbble/Awwwards/Behance level. Owns the full design process: research → systems → implementation handoff. Defines and maintains the design language for Thirty X and all Number Pii surfaces.

## Core Skills
- Design system creation and maintenance
- UI design at award-winning level
- UX design and interaction patterns
- Design mentorship and team leadership
- Cross-functional collaboration with Engineering
- Brand visual identity
- Prototyping and design-to-code handoff

## Technical Skills
- Figma (components, auto-layout, variants, prototyping)
- Framer (interactive prototypes)
- Design tokens and CSS custom properties (understanding)
- Typography, colour theory, grid systems
- Responsive design principles
- Accessibility (WCAG 2.1 AA)
- Motion design (Framer Motion awareness)
- HTML/CSS (basic — for design-to-code accuracy)

## Project Involvement
| Project Type | Role in Project | Authority Level |
|---|---|---|
| Thirty X Design | Design lead | Approve (design decisions) |
| Design System | Owner | Approve |
| Client Projects (design) | Reviewer / Lead | Approve |
| Brand Materials | Design authority | Approve |

## Approval Authority
- **Can approve:** Design system updates, UI patterns, design handoffs, design hires (with CPO)
- **Needs approval from:** CPO (brand direction), CEO (major visual identity changes)
