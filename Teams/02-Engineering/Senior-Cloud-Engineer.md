# Senior Cloud Engineer

## Position Details
- **Department:** Engineering — DevOps & Infrastructure
- **Reports To:** Head of DevOps & Infrastructure
- **Direct Reports:** None
- **Employment Type:** Full-time

## Role Summary
Specialises in cloud architecture, cost optimisation, and multi-environment infrastructure. Designs scalable, secure, and cost-effective cloud solutions.

## Core Skills
- Cloud architecture design (@cloud-architect, @software-architecture)
- Multi-environment management (dev/staging/prod) (@cloud-devops, @deployment-procedures)
- Cost optimisation and resource right-sizing (@aws-cost-optimizer, @cost-optimization)
- Network architecture and security groups (@cloud-architect, @secrets-management)
- Disaster recovery and backup strategies (@cloud-architect, @aws-skills)

## Technical Skills
- AWS (VPC, Route 53, CloudFront, IAM, GuardDuty, Config) (@aws-skills)
- Terraform, CloudFormation (@terraform-infrastructure, @cloudformation-best-practices)
- Kubernetes, ECS, serverless architectures (@kubernetes-deployment, @aws-serverless)
- Cloud cost management tools (AWS Cost Explorer, Infracost) (@aws-cost-optimizer)
- Cloud security (IAM policies, security groups, KMS) (@secrets-management, @k8s-security-policies)
- Multi-region and high-availability patterns (@cloud-architect)
