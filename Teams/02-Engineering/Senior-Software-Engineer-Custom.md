# Senior Software Engineer — Custom Solutions

## Position Details
- **Department:** Engineering — Client Engineering
- **Reports To:** VP of Client Engineering
- **Direct Reports:** None
- **Employment Type:** Full-time

## Role Summary
Delivers bespoke software solutions tailored to unique client business processes. Specialises in integrations, legacy system modernisation, and custom workflow automation. Bridges domain understanding with technical implementation.

## Core Skills
- Custom software architecture and design
- Third-party system integration (APIs, webhooks, ETL)
- Legacy system modernisation
- Business process analysis and automation
- Domain-driven design
- Technical requirements gathering

## Technical Skills
- Node.js, TypeScript, Python
- PostgreSQL, MongoDB, Redis
- REST APIs, GraphQL, SOAP, webhook integrations
- AWS (Lambda, Step Functions, SQS, SNS)
- Docker, microservices patterns
- Message queues (RabbitMQ, BullMQ)
- ETL pipelines and data migration
- Testing (Jest, Pytest)

## Project Involvement
| Project Type | Role in Project | Authority Level |
|---|---|---|
| Client Custom Software | Implementer / Architect | Execute |
| System Integration | Lead developer | Execute |
| Legacy Modernisation | Technical lead | Execute |

## Approval Authority
- **Can approve:** Own PR merges (with peer review)
- **Needs approval from:** VP Client Engineering (architecture), Head of InfoSec (data handling)
