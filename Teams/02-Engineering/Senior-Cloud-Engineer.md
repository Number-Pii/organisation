# Senior Cloud Engineer

## Position Details
- **Department:** Engineering — DevOps & Infrastructure
- **Reports To:** Head of DevOps & Infrastructure
- **Direct Reports:** None
- **Employment Type:** Full-time

## Role Summary
Specialises in cloud architecture, cost optimisation, and multi-environment infrastructure. Designs scalable, secure, and cost-effective cloud solutions.

## Core Skills
- Cloud architecture design
- Multi-environment management (dev/staging/prod)
- Cost optimisation and resource right-sizing
- Network architecture and security groups
- Disaster recovery and backup strategies

## Technical Skills
- AWS (VPC, Route 53, CloudFront, IAM, GuardDuty, Config)
- Terraform, CloudFormation
- Kubernetes, ECS, serverless architectures
- Cloud cost management tools (AWS Cost Explorer, Infracost)
- Cloud security (IAM policies, security groups, KMS)
- Multi-region and high-availability patterns

## Project Involvement
| Project Type | Role in Project | Authority Level |
|---|---|---|
| Cloud Architecture | Designer / Implementer | Execute |
| Cost Optimisation | Lead | Execute |
| New Environment Setup | Owner | Execute |

## Approval Authority
- **Can approve:** Environment configurations
- **Needs approval from:** Head of DevOps (provisioning), Head of InfoSec (security policies)
