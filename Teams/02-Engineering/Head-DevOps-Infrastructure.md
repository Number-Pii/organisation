# Head of DevOps & Infrastructure

## Position Details
- **Department:** Engineering
- **Reports To:** CTO
- **Direct Reports:** Senior DevOps Engineer(s), Senior Cloud Engineer
- **Employment Type:** Full-time

## Role Summary
Owns all infrastructure, deployment pipelines, and platform reliability. Has designed multi-region, high-availability infrastructure at scale. Ensures every product and client project ships on robust, secure, and scalable infrastructure.

## Core Skills
- Infrastructure architecture and design (@cloud-architect, @software-architecture)
- CI/CD pipeline design and optimisation (@cicd-automation-workflow-automate, @github-actions-templates)
- Cloud cost management and optimisation (@aws-cost-optimizer, @cost-optimization)
- High availability and disaster recovery (@cloud-architect, @aws-skills)
- Container orchestration (@kubernetes-architect, @docker-expert)
- Monitoring, alerting, and incident response (@grafana-dashboards, @pagerduty-automation)
- Infrastructure as Code (@terraform-specialist, @cloudformation-best-practices)

## Technical Skills
- AWS (EC2, ECS, Lambda, RDS, S3, CloudFront, Route 53, VPC) (@aws-skills, @aws-serverless)
- Docker, Kubernetes, ECS Fargate (@docker-expert, @kubernetes-architect)
- Terraform, CloudFormation (@terraform-specialist, @cloudformation-best-practices)
- GitHub Actions, GitLab CI, Jenkins (@github-actions-templates, @cicd-automation-workflow-automate)
- Vercel, Netlify (static deployments) (@vercel-deployment)
- Datadog, CloudWatch, PagerDuty (@grafana-dashboards, @pagerduty-automation)
- Nginx, load balancers, CDN configuration (@cloud-devops)
- Linux server administration (@linux-troubleshooting, @linux-shell-scripting)

- **Can approve:** Infrastructure changes, deployment pipeline modifications, cloud resource provisioning, DevOps hires
- **Needs approval from:** CTO (major architecture changes, large cost commitments)
