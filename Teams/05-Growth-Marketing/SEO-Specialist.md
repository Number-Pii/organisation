# SEO Specialist

## Position Details
- **Department:** Growth & Marketing
- **Reports To:** Head of Content & SEO
- **Direct Reports:** None
- **Employment Type:** Full-time

## Role Summary
Owns technical SEO, keyword strategy, and search performance tracking. Ensures the website and content rank competitively and drive qualified organic traffic.

## Core Skills
- Technical SEO (site structure, crawl optimisation, Core Web Vitals)
- Keyword research and competitive analysis
- On-page and off-page SEO
- Link building strategy
- Search performance reporting

## Technical Skills
- Ahrefs, SEMrush, Moz
- Google Search Console, Google Analytics 4
- Screaming Frog, Sitebulb
- Schema markup / JSON-LD
- Core Web Vitals monitoring
- HTML/CSS basics (for technical SEO fixes)

## Project Involvement
| Project Type | Role in Project | Authority Level |
|---|---|---|
| Technical SEO | Owner | Execute |
| Keyword Strategy | Lead | Execute |
| SEO Reporting | Owner | Execute |
| Website Performance | SEO advisor | Advise |

## Approval Authority
- **Can approve:** SEO recommendations, keyword targets
- **Needs approval from:** Head of Content (major strategy changes), CTO (technical SEO that requires code changes)
