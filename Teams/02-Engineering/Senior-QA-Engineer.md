# Senior QA Engineer

## Position Details
- **Department:** Engineering — QA & Reliability
- **Reports To:** Head of QA & Reliability
- **Direct Reports:** None
- **Employment Type:** Full-time

## Role Summary
Designs and executes test strategies for products and client deliveries. Writes test plans, performs manual exploratory testing, and identifies regressions.

## Core Skills
- Test planning and execution
- Exploratory and regression testing
- Bug reporting and severity classification
- UAT coordination
- Cross-browser and cross-device testing

## Technical Skills
- Playwright, Cypress (E2E)
- Jest, Testing Library (unit/component)
- Postman (API testing)
- Lighthouse (performance)
- JIRA, Linear (bug tracking)
- Git, CI/CD awareness
- Mobile device testing

## Project Involvement
| Project Type | Role in Project | Authority Level |
|---|---|---|
| Product Releases | Tester / Quality reviewer | Execute |
| Client Deliveries | Quality gate | Advise |

## Approval Authority
- **Can approve:** Test case sign-off, bug severity classification
- **Needs approval from:** Head of QA (release decisions)
