# Senior Security Engineer

## Position Details
- **Department:** Engineering — Information Security
- **Reports To:** Head of Information Security
- **Direct Reports:** None
- **Employment Type:** Full-time

## Role Summary
Implements security controls, conducts code security reviews, performs penetration testing, and manages vulnerability remediation across all products and client deliveries.

## Core Skills
- Application security testing (SAST/DAST) (@security-scanning-security-sast, @web-security-testing)
- Penetration testing and vulnerability assessment (@pentest-checklist, @ethical-hacking-methodology)
- Secure code review (@security-audit, @code-review-excellence)
- Infrastructure security hardening (@security-scanning-security-hardening, @cloud-architect)
- Incident response execution (@pagerduty-automation, @security-audit)
- Security tool management (@vulnerability-scanner, @scanning-tools)

## Technical Skills
- OWASP Top 10, secure coding practices (@web-security-testing, @security-audit)
- Burp Suite, OWASP ZAP (web app security testing) (@burp-suite-testing)
- Snyk, SonarQube, Checkmarx (SAST) (@security-scanning-security-sast)
- AWS security (IAM, Security Hub, GuardDuty, WAF) (@aws-skills, @secrets-management)
- Docker/container security scanning (Trivy, Grype) (@k8s-security-policies, @docker-expert)
- Network security (firewalls, VPC, security groups) (@cloud-architect, @security-scanning-security-hardening)
- Authentication/authorisation protocols (OAuth 2.0, JWT) (@auth-implementation-patterns)
- Python, Bash (security tooling and automation) (@python-pro, @linux-shell-scripting)
- SIEM tools, log analysis (@grafana-dashboards, @security-audit)
