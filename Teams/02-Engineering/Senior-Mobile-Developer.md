# Senior Mobile Developer

## Position Details
- **Department:** Engineering — Client Engineering
- **Reports To:** Lead Mobile Developer
- **Direct Reports:** None
- **Employment Type:** Full-time

## Role Summary
Builds mobile applications for client projects across iOS and Android. Expert in cross-platform frameworks with the ability to drop into native code when needed.

## Core Skills
- Cross-platform mobile development
- Native platform knowledge (iOS + Android)
- Mobile UI/UX implementation
- App performance optimisation
- App store submission and release management

## Technical Skills
- React Native, Expo
- Flutter, Dart
- Swift / Kotlin (native when needed)
- Mobile APIs (push, camera, GPS, biometrics)
- Firebase, REST API integration
- Detox, Appium (mobile testing)
- Fastlane, EAS Build (CI/CD)
- Git, code review workflows

## Project Involvement
| Project Type | Role in Project | Authority Level |
|---|---|---|
| Client Mobile App | Implementer | Execute |
| Code Reviews | Reviewer / Reviewee | Execute |

## Approval Authority
- **Can approve:** Own PR merges (with peer review)
- **Needs approval from:** Lead Mobile Developer (platform choice, architectural decisions)
