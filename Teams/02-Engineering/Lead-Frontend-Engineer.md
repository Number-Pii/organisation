# Lead Frontend Engineer

## Position Details
- **Department:** Engineering — Product Engineering
- **Reports To:** VP of Product Engineering
- **Direct Reports:** Senior Frontend Engineer(s)
- **Employment Type:** Full-time

## Role Summary
The technical lead for all frontend engineering on Number Pii products. Owns the frontend architecture, design system implementation, component library, and performance standards for Thirty X and future products.

## Core Skills
- Frontend architecture and design system implementation
- Component-driven development
- Performance optimisation (Core Web Vitals, Lighthouse)
- Accessibility (WCAG 2.1 AA)
- Code review and mentorship
- Cross-browser compatibility
- Animation and interaction design implementation

## Technical Skills
- React, Next.js (App Router), TypeScript
- CSS Modules, CSS Custom Properties, responsive design
- Framer Motion (scroll animations, layout transitions)
- HTML5 semantic markup, ARIA attributes
- Jest, Vitest, Testing Library, Playwright
- Webpack, Turbopack, ESBuild
- Storybook (component documentation)
- Git, GitHub, code review workflows
- Lighthouse, WebPageTest, Chrome DevTools
- Figma (design-to-code handoff)

## Project Involvement
| Project Type | Role in Project | Authority Level |
|---|---|---|
| Thirty X Frontend | Technical lead | Approve (frontend decisions) |
| Design System | Owner (implementation) | Approve |
| Frontend Code Reviews | Reviewer | Approve |
| Performance Audits | Lead | Execute |

## Approval Authority
- **Can approve:** Frontend architectural decisions, component patterns, PR merges
- **Needs approval from:** VP Product Engineering (major refactors), Lead Product Designer (design deviations)
