# Senior Web Developer

## Position Details
- **Department:** Engineering — Client Engineering
- **Reports To:** Lead Web Developer
- **Direct Reports:** None
- **Employment Type:** Full-time

## Role Summary
Delivers web application features for client projects to an exceptional standard. Works within project architectures set by the Lead Web Developer. Capable of owning feature domains end-to-end on client engagements.

## Core Skills
- Web application development
- Responsive and accessible UI implementation
- API integration and data handling
- Client project delivery to deadlines
- Code quality and testing

## Technical Skills
- React, Next.js, TypeScript, Vue.js
- Node.js, Express, PostgreSQL
- CSS/SCSS, CSS Modules, responsive design
- REST APIs, GraphQL, CMS integrations
- Jest, Playwright, Cypress
- Git, CI/CD, Vercel/AWS deployment

## Project Involvement
| Project Type | Role in Project | Authority Level |
|---|---|---|
| Client Website / Web App | Implementer | Execute |
| Code Reviews | Reviewer / Reviewee | Execute |

## Approval Authority
- **Can approve:** Own PR merges (with peer review)
- **Needs approval from:** Lead Web Developer (architectural changes)
