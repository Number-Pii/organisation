# Senior Frontend Engineer

## Position Details
- **Department:** Engineering — Product Engineering
- **Reports To:** Lead Frontend Engineer
- **Direct Reports:** None
- **Employment Type:** Full-time

## Role Summary
Implements features, components, and pages for Number Pii products. Works within the established design system and architectural patterns. Capable of independently owning entire feature domains from design handoff to production.

## Core Skills
- Component development and state management
- Responsive and accessible UI implementation
- Animation and micro-interaction development
- Performance optimisation
- Unit and integration testing
- Code review participation

## Technical Skills
- React, Next.js (App Router), TypeScript
- CSS Modules, CSS Custom Properties, clamp(), media queries
- Framer Motion (animations, scroll-linked transforms)
- React Hook Form, Zod (form validation)
- Jest, Vitest, Testing Library
- Git, GitHub pull request workflows
- Figma (extracting design specs)
- Chrome DevTools, React DevTools
