# SEO Specialist

## Position Details
- **Department:** Growth & Marketing
- **Reports To:** Head of Content & SEO
- **Direct Reports:** None
- **Employment Type:** Full-time

## Role Summary
Owns technical SEO, keyword strategy, and search performance tracking. Ensures the website and content rank competitively and drive qualified organic traffic.

## Core Skills
- Technical SEO (site structure, crawl optimisation, Core Web Vitals) (@seo-technical, @web-performance-optimization)
- Keyword research and competitive analysis (@seo-keyword-strategist, @competitive-landscape)
- On-page and off-page SEO (@seo-page, @seo-content)
- Link building strategy (@seo-authority-builder)
- Search performance reporting (@analytics-product, @google-analytics-automation)

## Technical Skills
- Ahrefs, SEMrush, Moz (@seo-audit, @seo-keyword-strategist)
- Google Search Console, Google Analytics 4 (@google-analytics-automation, @analytics-tracking)
- Screaming Frog, Sitebulb (@seo-technical, @seo-audit)
- Schema markup / JSON-LD (@seo-schema, @schema-markup)
- Core Web Vitals monitoring (@web-performance-optimization)
- HTML/CSS basics (for technical SEO fixes) (@seo-technical, @frontend-dev-guidelines)

## Project Involvement
| Project Type | Role in Project | Authority Level |
|---|---|---|

- **Can approve:** SEO recommendations, keyword targets
