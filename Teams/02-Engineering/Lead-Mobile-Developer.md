# Lead Mobile Developer

## Position Details
- **Department:** Engineering — Client Engineering
- **Reports To:** VP of Client Engineering
- **Direct Reports:** Senior Mobile Developer(s)
- **Employment Type:** Full-time

## Role Summary
Leads client-facing mobile development projects. Expert in native and cross-platform mobile development for iOS and Android. Owns mobile architecture, performance, and app store delivery processes for client engagements.

## Core Skills
- Mobile application architecture
- Cross-platform development strategy
- App Store and Google Play submission processes
- Mobile performance optimisation
- Client communication and technical presentation
- Team leadership and code review

## Technical Skills
- React Native, Expo
- Flutter, Dart
- Swift / SwiftUI (iOS native)
- Kotlin / Jetpack Compose (Android native)
- Mobile-specific APIs (push notifications, camera, GPS, biometrics)
- REST API integration from mobile
- Firebase (auth, messaging, analytics)
- App store deployment pipelines
- Mobile testing (Detox, Appium)
- Git, CI/CD for mobile (Fastlane, EAS)

## Project Involvement
| Project Type | Role in Project | Authority Level |
|---|---|---|
| Client Mobile App | Project lead | Approve (technical decisions) |
| Mobile Scoping | Estimation lead | Execute |
| Cross-Platform Strategy | Advisor | Advise |
| Mobile Code Reviews | Reviewer | Approve |

## Approval Authority
- **Can approve:** Mobile project technical decisions, platform choice, team assignments, PR merges
- **Needs approval from:** VP Client Engineering (scope/budget changes)
