# Lead Web Developer

## Position Details
- **Department:** Engineering — Client Engineering
- **Reports To:** VP of Client Engineering
- **Direct Reports:** Senior Web Developer(s)
- **Employment Type:** Full-time

## Role Summary
Leads client-facing web development projects from scoping to delivery. Capable of independently architecting and delivering a full web application engagement. Client-facing confidence — presents technical decisions to non-technical stakeholders.

## Core Skills
- Web application architecture for client projects (@software-architecture, @senior-architect)
- Project scoping and technical estimation (@writing-plans, @project-development)
- Client communication and presentation (@customer-support, @internal-comms)
- Team coordination and delegation (@project-development, @team-collaboration-standup-notes)
- Code review and quality enforcement (@code-review-excellence, @frontend-dev-guidelines)
- Multi-project management (@project-development)

## Technical Skills
- React, Next.js, TypeScript, Vue.js (@react-best-practices, @nextjs-best-practices, @typescript-expert)
- Node.js, Express, PostgreSQL, MongoDB (@nodejs-best-practices, @postgresql)
- CSS/SCSS, responsive design, CSS Modules (@tailwind-patterns, @web-design-guidelines)
- REST APIs, GraphQL, CMS integrations (Contentful, Sanity, Strapi) (@api-design-principles)
- Vercel, AWS, Netlify (deployment) (@vercel-deployment, @aws-skills)
