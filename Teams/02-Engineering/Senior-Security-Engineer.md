# Senior Security Engineer

## Position Details
- **Department:** Engineering — Information Security
- **Reports To:** Head of Information Security
- **Direct Reports:** None
- **Employment Type:** Full-time

## Role Summary
Implements security controls, conducts code security reviews, performs penetration testing, and manages vulnerability remediation across all products and client deliveries.

## Core Skills
- Application security testing (SAST/DAST)
- Penetration testing and vulnerability assessment
- Secure code review
- Infrastructure security hardening
- Incident response execution
- Security tool management

## Technical Skills
- OWASP Top 10, secure coding practices
- Burp Suite, OWASP ZAP (web app security testing)
- Snyk, SonarQube, Checkmarx (SAST)
- AWS security (IAM, Security Hub, GuardDuty, WAF)
- Docker/container security scanning (Trivy, Grype)
- Network security (firewalls, VPC, security groups)
- Authentication/authorisation protocols (OAuth 2.0, JWT)
- Python, Bash (security tooling and automation)
- SIEM tools, log analysis

## Project Involvement
| Project Type | Role in Project | Authority Level |
|---|---|---|
| Security Audits | Executor | Execute |
| Penetration Tests | Tester | Execute |
| Code Security Reviews | Reviewer | Advise |
| Incident Response | Responder | Execute |

## Approval Authority
- **Can approve:** Vulnerability severity ratings, remediation priorities
- **Needs approval from:** Head of InfoSec (policy changes, major findings disclosure)
