# Senior Product Designer

## Position Details
- **Department:** Product & Design
- **Reports To:** Lead Product Designer
- **Direct Reports:** None
- **Employment Type:** Full-time

## Role Summary
Designs features, flows, and interfaces for Thirty X and client projects within the established design system. Delivers production-ready designs with specifications for engineering handoff.

## Core Skills
- UI/UX design
- Wireframing and prototyping
- Design system usage and contribution
- User flow mapping
- Design handoff and developer collaboration

## Technical Skills
- Figma (design, prototyping, dev mode)
- Responsive and mobile-first design
- Accessibility best practices
- Basic understanding of React/CSS for handoff accuracy

## Project Involvement
| Project Type | Role in Project | Authority Level |
|---|---|---|
| Product Features | Designer | Execute |
| Client Projects | Designer | Execute |
| Design System | Contributor | Execute |

## Approval Authority
- **Can approve:** Own design deliverables (with Lead review)
- **Needs approval from:** Lead Product Designer (new patterns, system changes)
