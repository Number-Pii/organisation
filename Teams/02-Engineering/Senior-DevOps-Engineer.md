# Senior DevOps Engineer

## Position Details
- **Department:** Engineering — DevOps & Infrastructure
- **Reports To:** Head of DevOps & Infrastructure
- **Direct Reports:** None
- **Employment Type:** Full-time

## Role Summary
Builds and maintains CI/CD pipelines, manages cloud infrastructure, and ensures deployments are automated, reliable, and secure.

## Core Skills
- CI/CD pipeline implementation and maintenance
- Infrastructure as Code
- Container management and orchestration
- Monitoring and alerting setup
- Incident response and troubleshooting

## Technical Skills
- AWS (EC2, ECS, Lambda, RDS, S3, CloudFront)
- Docker, Kubernetes, ECS Fargate
- Terraform, CloudFormation
- GitHub Actions, GitLab CI
- Datadog, CloudWatch, PagerDuty
- Nginx, load balancers
- Linux administration, shell scripting
- Vercel, Netlify

## Project Involvement
| Project Type | Role in Project | Authority Level |
|---|---|---|
| Infrastructure Changes | Implementer | Execute |
| Deployment Pipelines | Builder / Maintainer | Execute |
| Incident Response | Responder | Execute |

## Approval Authority
- **Can approve:** Pipeline configuration changes
- **Needs approval from:** Head of DevOps (infrastructure provisioning, cost changes)
