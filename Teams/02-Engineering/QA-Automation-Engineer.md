# QA Automation Engineer

## Position Details
- **Department:** Engineering — QA & Reliability
- **Reports To:** Head of QA & Reliability
- **Direct Reports:** None
- **Employment Type:** Full-time

## Role Summary
Builds and maintains automated test suites, CI/CD test pipelines, and quality infrastructure. Reduces manual testing burden by automating regression, performance, and integration tests.

## Core Skills
- Test automation framework development
- CI/CD test pipeline integration
- Performance and load testing
- Test data management
- Flaky test detection and maintenance

## Technical Skills
- Playwright, Cypress (E2E automation)
- Jest, Vitest (unit/integration automation)
- k6, Artillery (load testing)
- GitHub Actions (CI test pipelines)
- Docker (test environments)
- TypeScript, JavaScript
- API testing automation (Supertest)
- Lighthouse CI (performance regression)

## Project Involvement
| Project Type | Role in Project | Authority Level |
|---|---|---|
| Test Infrastructure | Builder / Maintainer | Execute |
| CI/CD Quality Gates | Owner | Execute |
| Performance Benchmarks | Runner | Execute |

## Approval Authority
- **Can approve:** Test automation configuration, pipeline test stages
- **Needs approval from:** Head of QA (test strategy changes)
