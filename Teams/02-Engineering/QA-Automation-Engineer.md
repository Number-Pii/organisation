# QA Automation Engineer

## Position Details
- **Department:** Engineering — QA & Reliability
- **Reports To:** Head of QA & Reliability
- **Direct Reports:** None
- **Employment Type:** Full-time

## Role Summary
Builds and maintains automated test suites, CI/CD test pipelines, and quality infrastructure. Reduces manual testing burden by automating regression, performance, and integration tests.

## Core Skills
- Test automation framework development (@e2e-testing, @test-automator)
- CI/CD test pipeline integration (@cicd-automation-workflow-automate, @github-actions-templates)
- Performance and load testing (@k6-load-testing, @performance-testing-review-ai-review)
- Test data management (@testing-qa, @database-design)
- Flaky test detection and maintenance (@systematic-debugging, @testing-patterns)

## Technical Skills
- Playwright, Cypress (E2E automation) (@playwright-skill, @e2e-testing)
- Jest, Vitest (unit/integration automation) (@javascript-testing-patterns, @tdd-workflows-tdd-cycle)
- k6, Artillery (load testing) (@k6-load-testing)
- GitHub Actions (CI test pipelines) (@github-actions-templates, @cicd-automation-workflow-automate)
- Docker (test environments) (@docker-expert)
- TypeScript, JavaScript (@typescript-expert, @javascript-testing-patterns)
- API testing automation (Supertest) (@api-design-principles, @javascript-testing-patterns)
- Lighthouse CI (performance regression) (@web-performance-optimization)
