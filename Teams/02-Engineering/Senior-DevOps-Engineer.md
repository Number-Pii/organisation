# Senior DevOps Engineer

## Position Details
- **Department:** Engineering — DevOps & Infrastructure
- **Reports To:** Head of DevOps & Infrastructure
- **Direct Reports:** None
- **Employment Type:** Full-time

## Role Summary
Builds and maintains CI/CD pipelines, manages cloud infrastructure, and ensures deployments are automated, reliable, and secure.

## Core Skills
- CI/CD pipeline implementation and maintenance (@cicd-automation-workflow-automate, @github-actions-templates)
- Infrastructure as Code (@terraform-specialist, @cloudformation-best-practices)
- Container management and orchestration (@docker-expert, @kubernetes-deployment)
- Monitoring and alerting setup (@grafana-dashboards, @pagerduty-automation)
- Incident response and troubleshooting (@linux-troubleshooting, @cloud-devops)

## Technical Skills
- AWS (EC2, ECS, Lambda, RDS, S3, CloudFront) (@aws-skills, @aws-serverless)
- Docker, Kubernetes, ECS Fargate (@docker-expert, @kubernetes-deployment)
- Terraform, CloudFormation (@terraform-specialist, @cloudformation-best-practices)
- GitHub Actions, GitLab CI (@github-actions-templates, @cicd-automation-workflow-automate)
- Datadog, CloudWatch, PagerDuty (@grafana-dashboards, @pagerduty-automation)
- Nginx, load balancers (@cloud-devops)
- Linux administration, shell scripting (@linux-troubleshooting, @linux-shell-scripting)
- Vercel, Netlify (@vercel-deployment)
