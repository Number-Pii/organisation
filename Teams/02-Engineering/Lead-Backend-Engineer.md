# Lead Backend Engineer

## Position Details
- **Department:** Engineering — Product Engineering
- **Reports To:** VP of Product Engineering
- **Direct Reports:** Senior Backend Engineer(s)
- **Employment Type:** Full-time

## Role Summary
The technical lead for all backend engineering on Number Pii products. Owns API design, database architecture, data models, and backend performance for Thirty X and future products.

## Core Skills
- Backend architecture and API design (@backend-dev-guidelines, @api-design-principles)
- Database design and optimisation (@database-design, @postgresql)
- Authentication and authorisation systems (@auth-implementation-patterns)
- Microservices and event-driven architecture (@microservices-patterns)
- Code review and mentorship (@code-review-excellence)
- Data migration and versioning strategies (@database-migration)
- Performance profiling and caching strategies (@performance-profiling)

## Technical Skills
- Node.js, Express, Fastify, NestJS (@nestjs-expert, @nodejs-best-practices)
- PostgreSQL (schema design, query optimisation, migrations) (@postgresql, @database-design)
- GraphQL, REST API design (@api-design-principles)
- Redis (caching, sessions, queues) (@backend-dev-guidelines)
- OAuth 2.0, JWT, OIDC (authentication) (@auth-implementation-patterns)
- Prisma, Drizzle ORM, Knex (@drizzle-orm-expert, @prisma-expert)
- Bull/BullMQ (job queues) (@bullmq-specialist)
- AWS (RDS, Lambda, SQS, SNS, S3) (@aws-skills, @aws-serverless)
- Jest, Supertest (API testing) (@javascript-testing-patterns, @tdd-workflows-tdd-cycle)
- Docker (containerised services) (@docker-expert)
