# Lead Web Developer

## Position Details
- **Department:** Engineering — Client Engineering
- **Reports To:** VP of Client Engineering
- **Direct Reports:** Senior Web Developer(s)
- **Employment Type:** Full-time

## Role Summary
Leads client-facing web development projects from scoping to delivery. Capable of independently architecting and delivering a full web application engagement. Client-facing confidence — presents technical decisions to non-technical stakeholders.

## Core Skills
- Web application architecture for client projects
- Project scoping and technical estimation
- Client communication and presentation
- Team coordination and delegation
- Code review and quality enforcement
- Multi-project management

## Technical Skills
- React, Next.js, TypeScript, Vue.js
- Node.js, Express, PostgreSQL, MongoDB
- CSS/SCSS, responsive design, CSS Modules
- REST APIs, GraphQL, CMS integrations (Contentful, Sanity, Strapi)
- Vercel, AWS, Netlify (deployment)
- Playwright, Cypress (E2E testing)
- WordPress, Shopify (when client requires)
- Git, GitHub, CI/CD

## Project Involvement
| Project Type | Role in Project | Authority Level |
|---|---|---|
| Client Website Build | Project lead | Approve (technical decisions) |
| Client Web App | Technical lead | Approve |
| Client Scoping | Estimation lead | Execute |
| Web Team Code Reviews | Reviewer | Approve |

## Approval Authority
- **Can approve:** Client web project technical decisions, team assignments within project, PR merges
- **Needs approval from:** VP Client Engineering (scope/budget changes), Account Manager (client-facing changes)
